Windows

Descripción

Windows, MS Windows o Microsoft Windows es una familia de sistemas operativos para computadores personales, teléfonos inteligentes y otros dispositivos, creada y comercializada por la empresa estadounidense Microsoft.

Más que un sistema operativo único, Windows comprende una serie de versiones derivadas de antiguos sistemas como MS-DOS y Windows NT. Estas versiones ofrecen una interfaz gráfica basada en un escritorio virtual, con elementos visuales como carpetas, archivos y una papelera de reciclaje.

La primera versión de Windows se lanzó en 1985, como una evolución del MS-DOS hacia entornos gráficos de usuario (GUI). Desde entonces, ha dominado el mercado mundial, alcanzando durante muchos años más del 90% de la cuota de sistemas operativos para PC.
